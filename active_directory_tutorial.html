Active Directory : Configuration et Gestion

Introduction

Active Directory (AD) est un service de gestion des identités et des accès, développé par Microsoft. Il est essentiel pour gérer les utilisateurs, les groupes, les ressources et les politiques dans un réseau d’entreprise.

Prérequis

  • Un serveur Windows Server configuré avec un accès administrateur.
  • Une compréhension de base des concepts réseaux, tels que les DNS et les IP.
  • Une version Windows Server prenant en charge Active Directory.

Étapes pour Configurer Active Directory

  1. Installer le rôle Active Directory :

    Ouvrez le gestionnaire de serveur, cliquez sur "Ajouter des rôles et des fonctionnalités", et sélectionnez "Services de domaine Active Directory".

  2. Promouvoir le serveur en contrôleur de domaine :

    Après l'installation, cliquez sur "Promouvoir ce serveur en contrôleur de domaine". Configurez un nouveau domaine ou rejoignez un domaine existant.

  3. Configurer les utilisateurs et les groupes :

    Utilisez l'outil "Utilisateurs et ordinateurs Active Directory" pour créer et gérer des utilisateurs et des groupes.

  4. Configurer les stratégies de groupe (GPO) :

    Accédez à l’éditeur de gestion des stratégies de groupe pour appliquer des politiques à des utilisateurs ou des groupes spécifiques.

Commandes Utiles

  • dsadd user : Ajouter un nouvel utilisateur.
  • dsadd group : Ajouter un nouveau groupe.
  • dsquery user : Rechercher des utilisateurs dans AD.
  • gpupdate /force : Mettre à jour les stratégies de groupe.

Meilleures Pratiques

  • Configurer des mots de passe forts et des politiques de verrouillage de compte.
  • Mettre en place une hiérarchie de groupes et d’unités organisationnelles claire.
  • Surveiller régulièrement les journaux d'événements pour détecter les activités suspectes.
  • Planifier des sauvegardes régulières de l’Active Directory.
