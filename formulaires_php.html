Validation côté serveur

if (empty($nom)) { echo "Le champ Nom est obligatoire."; } if (!filter_var($email, FILTER_VALIDATE_EMAIL)) { echo "Email invalide."; }

Sécurisation des données

Utilisez des fonctions comme htmlspecialchars et filter_var pour prévenir les attaques XSS et autres vulnérabilités :

$nom = htmlspecialchars($_POST["nom"]); $email = filter_var($_POST["email"], FILTER_SANITIZE_EMAIL);
