Gestion des utilisateurs et permissions sur Linux

Introduction

La gestion des utilisateurs et des permissions est une compétence essentielle pour administrer un système Linux. Ce tutoriel explique les commandes et outils nécessaires pour gérer les utilisateurs et configurer les permissions de manière efficace.

Gestion des utilisateurs

Créer un utilisateur

Pour ajouter un nouvel utilisateur :

sudo adduser nom_utilisateur

Supprimer un utilisateur

Pour supprimer un utilisateur et son répertoire personnel :

sudo deluser --remove-home nom_utilisateur

Modifier un utilisateur

Pour changer des informations sur un utilisateur, utilisez :

sudo usermod -option nom_utilisateur

Exemple : Changer le nom d'utilisateur :

sudo usermod -l nouveau_nom ancien_nom

Gestion des groupes

Créer un groupe

sudo groupadd nom_groupe

Ajouter un utilisateur à un groupe

sudo usermod -aG nom_groupe nom_utilisateur

Supprimer un groupe

sudo groupdel nom_groupe

Gestion des permissions

Afficher les permissions

Pour voir les permissions d'un fichier ou d'un répertoire :

ls -l nom_fichier

Changer les permissions

Utilisez la commande chmod :

chmod mode nom_fichier

Exemple : Donner des permissions de lecture, écriture et exécution au propriétaire :

chmod 700 nom_fichier

Changer le propriétaire

Pour changer le propriétaire d'un fichier ou répertoire :

sudo chown utilisateur:group nom_fichier

Conseils pratiques

  • Utilisez sudo pour les commandes administratives.
  • Créez des groupes pour simplifier la gestion des permissions pour plusieurs utilisateurs.
  • Apprenez à utiliser les outils comme usermod et groupmod.
  • Évitez d'utiliser le compte root directement pour limiter les erreurs critiques.
