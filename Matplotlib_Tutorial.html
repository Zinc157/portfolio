Matplotlib : Création de visualisations

Introduction

Matplotlib est une bibliothèque de visualisation pour Python qui permet de créer des graphiques statiques, dynamiques et animés. Elle est utilisée pour représenter des données de manière claire et attrayante.

Installation

Pour installer Matplotlib, utilisez la commande suivante :

pip install matplotlib

Exemples de Graphiques

Graphique Linéaire

Voici comment créer un simple graphique linéaire :

import matplotlib.pyplot as plt x = [1, 2, 3, 4, 5] y = [2, 3, 5, 7, 11] plt.plot(x, y) plt.title("Graphique Linéaire") plt.xlabel("X") plt.ylabel("Y") plt.show()

Histogramme

Pour créer un histogramme :

import matplotlib.pyplot as plt data = [1, 1, 2, 3, 3, 3, 4, 4, 5, 5, 5, 5, 6] plt.hist(data, bins=6, color='blue', edgecolor='black') plt.title("Histogramme") plt.xlabel("Valeurs") plt.ylabel("Fréquence") plt.show()

Graphique en Secteurs

Un exemple de graphique en secteurs :

import matplotlib.pyplot as plt labels = ['A', 'B', 'C', 'D'] sizes = [15, 30, 45, 10] colors = ['gold', 'yellowgreen', 'lightcoral', 'lightskyblue'] plt.pie(sizes, labels=labels, colors=colors, autopct='%1.1f%%', startangle=140) plt.axis('equal') # Assure que le graphique est circulaire plt.title("Graphique en Secteurs") plt.show()
