Tutoriel : Mise en place et configuration d’Asterisk

Introduction

Asterisk est un système open-source puissant qui permet de créer des solutions de téléphonie VoIP personnalisées. Ce tutoriel explique comment installer et configurer Asterisk pour répondre aux besoins spécifiques d’un réseau.

Prérequis

  • Un serveur ou une machine virtuelle avec un système d'exploitation Linux (par exemple, CentOS ou Ubuntu).
  • Accès root à la machine.
  • Connaissance de base des commandes Linux.

Installation d’Asterisk

# Mettre à jour les paquets existants sudo apt update && sudo apt upgrade -y # Installer les dépendances nécessaires sudo apt install build-essential wget curl libncurses5-dev libssl-dev libxml2-dev -y # Télécharger les sources d’Asterisk cd /usr/src sudo wget http://downloads.asterisk.org/pub/telephony/asterisk/asterisk-18-current.tar.gz # Extraire les fichiers téléchargés sudo tar xvf asterisk-18-current.tar.gz cd asterisk-18.* # Installer les paquets supplémentaires sudo contrib/scripts/install_prereq install # Compiler et installer Asterisk sudo ./configure sudo make sudo make install sudo make samples sudo make config

Une fois ces étapes terminées, Asterisk est installé sur votre machine et prêt à être configuré.

Configuration de base

1. Configurer le fichier sip.conf

# Modifier le fichier sip.conf sudo nano /etc/asterisk/sip.conf # Exemple de configuration [general] disallow=all allow=ulaw defaultuser=utilisateur secret=motdepasse host=dynamic context=default

Ce fichier configure les paramètres SIP pour les utilisateurs.

2. Configurer le fichier extensions.conf

# Modifier le fichier extensions.conf sudo nano /etc/asterisk/extensions.conf # Exemple de configuration [default] exten => 100,1,Dial(SIP/100) exten => 101,1,Dial(SIP/101)

Ce fichier définit les extensions et leur comportement dans le système téléphonique.

Démarrage d’Asterisk

# Lancer le service Asterisk sudo systemctl start asterisk # Vérifier le statut sudo systemctl status asterisk # Accéder à la console CLI d’Asterisk sudo asterisk -r

Utilisez la console CLI pour tester et administrer Asterisk en temps réel.

Test et débogage

  • Vérifiez les utilisateurs enregistrés : sip show peers
  • Testez les appels entre extensions définies.
  • Consultez les journaux pour résoudre les erreurs : sudo tail -f /var/log/asterisk/messages
