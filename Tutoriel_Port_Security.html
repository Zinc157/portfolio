Tutoriel : Configuration de la sécurité des ports (Port Security)

Introduction

La fonctionnalité Port Security permet de renforcer la sécurité d'un réseau commuté en contrôlant l'accès aux ports d'un switch. Elle limite le nombre de périphériques pouvant se connecter à un port et peut détecter et prévenir les attaques, telles que le mac flooding.

Dans ce tutoriel, nous allons expliquer les concepts de base de la sécurité des ports et fournir des étapes pour configurer cette fonctionnalité sur des switches.

Concepts de base

  • Port Security : Une fonctionnalité qui permet de limiter le nombre d'adresses MAC autorisées sur un port.
  • Adresses MAC sécurisées :
    • Statique : Configurée manuellement par l'administrateur.
    • Dynamique : Apprise automatiquement par le switch et supprimée lors d'un redémarrage.
    • Collante (Sticky) : Apprise dynamiquement et sauvegardée dans la configuration en cours.
  • Modes de violation :
    • Protect : Les adresses MAC non autorisées sont ignorées (le port reste actif).
    • Restrict : Les adresses MAC non autorisées sont bloquées et une alerte est générée.
    • Shutdown : Le port est désactivé en cas de violation (par défaut).

Prérequis

  • Un switch compatible avec la fonctionnalité Port Security (Cisco, HP, etc.).
  • Compréhension de base des adresses MAC et de la configuration des switches.

Étapes pour configurer Port Security

Exemple avec des switches Cisco (IOS)

  1. Activer Port Security sur un port : 
    Switch> enable
Switch# configure terminal
Switch(config)# interface FastEthernet0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport port-security
  2. Configurer le nombre maximal d'adresses MAC autorisées : 
    Switch(config-if)# switchport port-security maximum 2
  3. Configurer le type d'adresses MAC sécurisées :
    • Adresses collantes (Sticky) : 
      Switch(config-if)# switchport port-security mac-address sticky
    • Adresses statiques : 
      Switch(config-if)# switchport port-security mac-address 00AA.BBCC.DDEE
  4. Définir l'action en cas de violation : 
    Switch(config-if)# switchport port-security violation shutdown
  5. Vérifier la configuration : 
    Switch# show port-security interface FastEthernet0/1

Commandes supplémentaires utiles

  • Vérifier l'état global de Port Security : 
    Switch# show port-security
  • Réinitialiser un port en cas de violation (mode shutdown) : 
    Switch# configure terminal
Switch(config)# interface FastEthernet0/1
Switch(config-if)# shutdown
Switch(config-if)# no shutdown
  • Supprimer la configuration de Port Security d'un port : 
    Switch(config-if)# no switchport port-security

Conseils pour une configuration optimale

  • Activez Port Security uniquement sur les ports connectés à des hôtes finaux.
  • Limitez le nombre d'adresses MAC à une seule sur les ports critiques (par exemple, imprimantes ou caméras IP).
  • Sauvegardez la configuration après avoir activé les adresses MAC collantes : 
    Switch# copy running-config startup-config

Conclusion

La sécurité des ports est une fonctionnalité essentielle pour protéger votre réseau contre les accès non autorisés et les attaques courantes. En suivant les étapes présentées, vous pouvez configurer Port Security sur vos switches pour améliorer la sécurité et la fiabilité de votre infrastructure réseau. Adaptez les commandes en fonction de vos besoins et de votre équipement.
