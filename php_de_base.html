Tutoriel : PHP de Base

Introduction

PHP est un langage de script côté serveur utilisé pour créer des pages web dynamiques. Il est facile à apprendre et largement utilisé dans le développement web.

Variables

Les variables en PHP commencent par un signe $. Par exemple :

$nom = "Jean"; $age = 30;

Boucles

PHP prend en charge plusieurs types de boucles, notamment :

  • for : Pour itérer un nombre fixe de fois.
  • while : Pour itérer tant qu'une condition est vraie.
  • foreach : Pour parcourir les tableaux.
// Exemple de boucle for for ($i = 0; $i < 5; $i++) { echo $i; }

Fonctions

Les fonctions permettent de regrouper du code réutilisable :

function direBonjour($nom) { return "Bonjour, $nom !"; } echo direBonjour("Alice");

Gestion des Erreurs

PHP dispose de plusieurs outils pour gérer les erreurs, comme try, catch, et finally :

try { $resultat = 10 / 0; } catch (Exception $e) { echo "Erreur : " . $e->getMessage(); } finally { echo "Opération terminée."; }
