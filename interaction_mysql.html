Tutoriel : Interaction avec MySQL

Introduction

PHP offre des fonctionnalités puissantes pour interagir avec des bases de données MySQL, permettant la gestion et l'utilisation dynamique des données dans vos applications web.

Connexion à une base de données

Pour se connecter à une base de données MySQL, utilisez l'extension mysqli ou PDO. Voici un exemple avec mysqli :

$servername = "localhost"; $username = "root"; $password = ""; $dbname = "ma_base"; $conn = new mysqli($servername, $username, $password, $dbname); if ($conn->connect_error) { die("Connexion échouée: " . $conn->connect_error); } echo "Connexion réussie";

Récupération de données

Pour récupérer des données d'une table, utilisez une requête SQL avec la méthode query :

$sql = "SELECT id, nom, email FROM utilisateurs"; $result = $conn->query($sql); if ($result->num_rows > 0) { while($row = $result->fetch_assoc()) { echo "ID: " . $row["id"] . " - Nom: " . $row["nom"] . " - Email: " . $row["email"] . "
"; } } else { echo "0 résultats"; }

Insertion de données

Pour insérer des données dans une table, utilisez une requête INSERT :

$sql = "INSERT INTO utilisateurs (nom, email) VALUES ('Jean Dupont', 'jean@example.com')"; if ($conn->query($sql) === TRUE) { echo "Nouvel enregistrement créé avec succès"; } else { echo "Erreur: " . $sql . "
" . $conn->error; }

Fermeture de la connexion

Une fois vos opérations terminées, fermez la connexion :

$conn->close();
