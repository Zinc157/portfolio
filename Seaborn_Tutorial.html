Seaborn : Visualisation de données statistiques

Introduction

Seaborn est une bibliothèque Python basée sur Matplotlib qui permet de créer des visualisations de données statistiques attractives et informatives avec un minimum de code. Cette bibliothèque est particulièrement utile pour explorer et comprendre des jeux de données complexes.

Installation

Pour utiliser Seaborn, vous devez l’installer avec pip :

pip install seaborn

Assurez-vous également d'avoir installé NumPy, Pandas et Matplotlib, car Seaborn dépend de ces bibliothèques.

Principales Fonctionnalités

  1. Visualisations univariées :

    Seaborn permet de créer facilement des histogrammes et des graphiques de densité.

    import seaborn as sns
import matplotlib.pyplot as plt

data = sns.load_dataset("iris")
sns.histplot(data['sepal_length'], kde=True)
plt.show()
  2. Visualisations bivariées :

    Créez des scatter plots et des heatmaps pour analyser les relations entre deux variables.

    sns.scatterplot(x='sepal_length', y='sepal_width', data=data)
plt.show()
  3. Graphiques catégoriels :

    Seaborn facilite la création de boxplots, barplots et autres graphiques catégoriels.

    sns.boxplot(x='species', y='sepal_length', data=data)
plt.show()

Personnalisation

Avec Seaborn, vous pouvez personnaliser vos graphiques à l’aide de thèmes prédéfinis :

sns.set_theme(style="whitegrid")
sns.boxplot(x='species', y='petal_length', data=data)
plt.show()

Les thèmes disponibles incluent darkgrid, whitegrid, dark, white, et ticks.
